## Start Devlopment

1. install npm https://www.npmjs.com/
2. 配置taobaoNPM镜像, 下载包速度快些 http://artifactory.hanyun.com/node/
3. cd platform-ui; npm install
4. npm -g install bower
5. npm -g install grunt-cli
6. bower install 
7. grunt live
